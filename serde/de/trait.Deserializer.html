Deserializer is a trait that can deserialize values by threading a Visitor trait through a value. It supports two entry point styles which enables different kinds of deserialization.

  1. The deserialize method. File formats like JSON embed the type of its construct in its file format. This allows the Deserializer to deserialize into a generic type like json::Value, which can represent all JSON types.

  2. The deserialize_* methods. File formats like bincode do not embed in its format how to decode its values. It relies instead on the Deserialize type to hint to the Deserializer with the deserialize_* methods how it should parse the next value. One downside though to only supporting the deserialize_* types is that it does not allow for deserializing into a generic json::Value-esque type.
